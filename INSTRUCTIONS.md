### Creating a repository from a template directly on Github

* Go to the https://github.com/orgs/SchweizerischeBundesbahnen/repositories and create a new repo
* At the top there is an option: Repository template, please choose SchweizerischeBundesbahnen/open-source-repo-template
* Select whether you want to include all branches, not just the default one. 
* The owner will always be the SchweizerischeBundesbahnen.
* Add a name to the new repo and an optional description.
* Set the visibility of a repository.
* Click the “Create repository” button and done!
